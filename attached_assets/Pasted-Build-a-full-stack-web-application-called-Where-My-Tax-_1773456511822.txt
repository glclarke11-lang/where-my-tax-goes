Build a full-stack web application called **“Where My Tax Goes”** that allows users to estimate how their personal taxes are distributed across government spending categories.

The application must be clean, modern, interactive, and data-visualization focused.

Tech stack:

* React (frontend)
* TailwindCSS (styling)
* Chart.js (data visualization)
* Node.js + Express backend
* JSON data file for government budget allocations
* Fully responsive design
* Dark mode support

The final product must be **fully functional**, not just UI mockups.

---

APP PURPOSE

Users enter their income or estimated tax paid.

The app calculates:

1. Estimated tax paid
2. Converts that tax into a **personalized government spending breakdown**
3. Displays the breakdown using **animated charts and interactive visuals**

Example:

User tax contribution: $20,000

Breakdown:

Healthcare — $7,200
Education — $3,000
Defence — $2,100
Infrastructure — $1,800
Welfare — $3,500
Government Admin — $1,200
Other — $1,200

Show this as:

• Animated pie chart
• Category cards with dollar values
• Percentage sliders

DATA STRUCTURE

Create a JSON file called **budgetData.json**

Example:

{
"healthcare": 0.36,
"education": 0.15,
"defence": 0.10,
"infrastructure": 0.09,
"welfare": 0.18,
"admin": 0.06,
"other": 0.06
}

This represents the default government spending distribution.
