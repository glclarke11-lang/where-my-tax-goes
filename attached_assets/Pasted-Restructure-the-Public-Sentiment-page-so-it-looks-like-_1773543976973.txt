Restructure the Public Sentiment page so it looks like a professional civic data dashboard.

Do not remove existing functionality. Reorganize the layout to improve clarity, credibility, and usability.

---

PAGE STRUCTURE

Divide the page into five sections:

1. Survey Overview
2. Participation Panel
3. Spending Preference Results
4. Government Comparison
5. Submit Your Preference

Each section should appear in this order from top to bottom.

---

SECTION 1 — SURVEY OVERVIEW

At the top of the page display a header card explaining the purpose of the page.

Example:

Title:
Public Budget Sentiment

Description:
See how Australians think tax money should be spent.

Display metadata fields:

Participants
Last updated
Confidence estimate (optional placeholder)

Example:

Participants: 3,482
Last updated: Today
Confidence estimate: ±3.5%

This section should look like a statistics panel.

Most Voted Category
